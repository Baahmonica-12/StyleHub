StyleHub - Online Shopping Application

StyleHub is a desktop-based e-commerce application built using "Java" and "JavaFX". The project features a modern GUI designed for a seamless shopping experience.

 Features
Product Browsing: View and search for various items.
Shopping Cart: Add items to a cart and view the total price.
Checkout System: Step-by-step checkout process.
Discount System: Integration for applying discount codes.

Technologies Used
Java: Core application logic.
JavaFX: For building the Graphical User Interface (GUI).
IntelliJ IDEA: Primary development environment.

 How to Run
 Clone the repository.
 Open the project in IntelliJ IDEA.
 Ensure the JavaFX library is correctly configured in your project settings.
 Run the main application file.
